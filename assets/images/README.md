## Optimasi Gambar

1. **Format yang disarankan:**

   - SVG untuk ikon dan ilustrasi
   - PNG untuk logo dengan transparansi
   - JPG untuk foto (quality 80-85%)
   - WebP sebagai alternatif modern

2. **Kompresi:**

   - Gunakan tools seperti TinyPNG atau ImageOptim
   - Target ukuran file < 100KB untuk gambar kecil
   - Target ukuran file < 500KB untuk gambar besar

3. **Responsive Images:**
   - Siapkan versi 1x, 2x, dan 3x untuk layar high-DPI
   - Gunakan format WebP dengan fallback JPG/PNG
